## 本地开发 & 阅读

本项目基于 vuepress 进行开发，以提供比 github mardown 更佳的阅读体验

依赖于 `node.js`、`yarn`、`vuepress` 等环境

```sh
#安装nodejs
#将yarn包路径添加到windows 环境变量
#安装vuepress
yarn global add vuepress

# 本地启动命令
yarn docs:dev

# 本地阅读
http://localhost:8080/doc-cleanarch/
```
